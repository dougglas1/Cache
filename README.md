### **Guidelines**

Version: net5.0

Have Docker installed.

Download Redis Image: ``` docker pull redis:7.0-rc2-alpine ```

Start a container with the image: ``` docker run -d -p 6379:6379 -i -t redis:7.0-rc2-alpine ```
